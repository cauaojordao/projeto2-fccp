# Fundamentos de Computação Concorrente, Paralela e Distribuida - Docker e Microsserviços

Este projeto contém 1 desafio implementado com .NET 8, demonstrando conceitos de Docker, volumes, Docker Compose e arquitetura de microsserviços.

## Estrutura do Projeto

```
projeto2-fccp/
├── desafio1/          # Containers em Rede
```

## Pré-requisitos

- Docker instalado
- Docker Compose instalado
- .NET 8 SDK (opcional, para desenvolvimento local)

## Como Executar

Cada desafio possui suas próprias instruções de execução. Navegue até a pasta do desafio desejado e siga as instruções no README específico ou execute os scripts `.sh` fornecidos.

## Stack Tecnológica

- **.NET 8**: Framework principal
- **Minimal APIs**: Para criação de endpoints HTTP
- **Docker**: Containerização
- **Docker Compose**: Orquestração de serviços
